This aligns with Go's standard marshalling behavior for anonymous fields and improves code organization. Key changes: - Refactored Marshal/Unmarshal methods to extract field processing into helper methods (marshalStruct/unmarshalStruct) - Added recursive traversal for anonymous embedded structs without index tags - Fields within anonymous embedded structs are now discovered and processed based on their own index tags - Maintains backward compatibility - existing behavior unchanged - Added comprehensive tests covering various scenarios This enables better code organization by allowing related ISO8583 fields to be grouped into logical Go structs without requiring manual flattening.
